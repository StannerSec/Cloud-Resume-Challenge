SIEM Home Lab

Started: July 2024

The SIEM Home Lab project aims to simulate a small-scale security operations center (SOC) environment at home. It includes setting up security information and event management systems to monitor and analyze network security incidents and events.

Technologies used: Elastic Stack (Elasticsearch, Logstash, Kibana), Docker, Suricata, Zeek
