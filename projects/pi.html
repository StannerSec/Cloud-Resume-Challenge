Raspberry Pi homelab

This project was created to display my knowledge gained during my studies for the Network+ and Security+ exams. Not only was it used during my study process, but also used as a leisure activity to build something fun.

Technologies used: Two Raspberry Pi 4's with 8 GB of ram, a 5 port TP link switch, and a Linksys router using 802.11 AX.
